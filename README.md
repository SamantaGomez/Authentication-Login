### Comandos usados
Ejecutaremos los siguientes comandos :
```
- Instalar ionic 
npm install -g @ionic/cli

- Ejecutar Ionnic
ionic serve
ionic serve -l

- Crear el servicio
ionic g service services/nombreservicio

- Instalar Firebase
npm install firebase @angular/fire --save

- Intalar Cordova
npm install -g ionic cordova

- Generar APK 
ionic cordova build android


```
